# 🎓 FCAI GPA Calculator Extension

A lightweight Chrome Extension that automatically calculates the cumulative GPA for students of the **Faculty of Computers and Artificial Intelligence – Cairo University** using the official FCAI grading system.

---

## 🚀 Release

**Current Version:** `v0.2`

### What's New
- ✅ Improved GPA calculation accuracy.
- ✅ Correct handling of repeated courses.
- ✅ Better popup loading experience.
- ✅ More reliable communication between popup and content scripts.
- ✅ Various bug fixes and stability improvements.

# 📦 Installation

## Option 1 (Recommended)

Download the latest release from the **Releases** page.

Then extract the ZIP file.

---

## Option 2 (Clone the Repository)

```bash
git clone https://github.com/EngOmarAbd-Elaziz/GPA_Calc_Extension.git
```

---

## Enable Developer Mode

Open

```
chrome://extensions
```

or

- Click the three dots (⋮)
- More Tools
- Extensions

<img width="1919" height="791" alt="Developer Mode" src="https://github.com/user-attachments/assets/5adbdc5f-6f94-4dae-82cf-efff9f8a269f" />

---

## Load the Extension

1. Click **Load unpacked**.
2. Select the extracted extension folder.
3. Open the FCAI Courses page.
4. Click the extension icon.
5. Your cumulative GPA will be calculated automatically.

# 📖 GPA Calculation

The extension calculates the cumulative GPA according to the **official FCAI Cairo University academic regulations**, including:

- ✔ Credit-hour weighted GPA
- ✔ Repeated course handling
- ✔ Failed course replacement
- ✔ Excluded zero-credit courses
- ✔ Automatic calculation directly from the university portal
